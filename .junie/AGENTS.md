# Project Guidelines

- Chaque code généré doit être systématiquement accompagné de tests unitaires également générés.
- La tâche ne finit pas tant que l'exécution de ces tests ou des tests impactés par le code modifié ne sont pas valides.
- Les tests doivent être placés dans le même répertoire que le fichier testé, avec le suffixe `.test.ts` (ou `.test.tsx` pour React).
- Utiliser `jest` et `@testing-library/react` comme frameworks de test (déjà configurés dans le projet).
